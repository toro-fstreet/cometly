# THMC Cometly Reports

Historical ad performance dashboard archive for The Hard Money Co.

- **Live index:** https://toro-fstreet.github.io/cometly/
- **Source:** Cometly (Meta + Google Ads)
- **Generated by:** Toro-Bot

## Structure

```
reports/
  YYYY-MM-DD/
    index.html   — full dashboard
    data.json    — raw scraped data
  history.csv    — aggregate metrics across all runs
index.html       — report browser
```

## Usage

Run `node cometly-scrape.js [days]` followed by `node cometly-build-dashboard.js` to generate a new report, then push to this repo.
